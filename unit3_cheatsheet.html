What are Javascript Objects?

Objects are just data, with added properties and methods.
Object Properties and Methods
Properties are values associated with objects.
Methods are actions objects can perform.

Object Properties Methods
Person person.name person.walk()
person.height person.eat()
person.weight person.talk()
person.country person.drink()

All persons have the same properties, but the property values differ from person to person.
All persons have the same methods, but they are performed at different times.
In JavaScript, objects are data (variables), with properties and methods.
Almost "everything" in JavaScript are treated as objects. Dates, Arrays, Strings, Functions...

Object Creation

In JavaScript you can also create your own objects. For example:

1. By object literal:

      var person = {name:"Daniel", height:"5'4\'", weight:120, country:"USA"};

2. By javascript keyword new:

      var person = new Object();
        person.name = "Daniel";
        person.height = "5'4\"";
        person.weight = 120;
        person.country = "USA";

3. By constructor:

The examples above are limited in many situations. They only create a single object.
Sometimes we like to have an "object type" that can be used to create many objects of one type.
The standard way to create an "object type" is to use an object.
      function person(name, height, weight, country) {
        this.name = name;
        this.height = height;
        this.weight = weight;
        this.country = country;
      }

      var dan = new person("Daniel", "5'4\"", 120, "USA!!!");

Adding a new property

You can add a new property to an existing object. For example: 
    dan.location = "San Francisco";

Adding a new method

You can add a new method to an existing object. For example: 
    dan.new_method = function (arg1, arg2) {
      code goes here;
    }

Accessing object properties

You can access existing properties of an object in 2 ways, like this: 
    dan.name;
    dan["name"];

Accessing object methods

You can call an object method like this: 
    dan.walk(50);
    dan.drink("milk");
