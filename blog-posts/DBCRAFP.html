 What are the differences between relative, absolute, and fixed positioning? #DBCRAFP

First off, all elements start out as static, meaning they don't have any default values but flow naturally onto a page.

With relative positioning, you add in positioning attributes and that would only affect the element/s relative to where it would be if it was static. So if you type in left: 10px, the element would position itself 10px to the left "relative" to where it would have been without that attribute (static).

With absolute positioning, you are able to place an element exactly anywhere on the page. You give it top, right, bottom, left position attributes, and the element will take that position of the page. These values will be relative to the next parent element with relative or absolute positioning.

Lastly, with fixed positioning, the element is positioned relative to the viewport, or the browser window itself. The viewport doesn't change when the window moves, so even if a user scrolls the page, the element will stay in position.
